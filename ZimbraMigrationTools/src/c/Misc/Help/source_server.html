Server Migration Source Information
You can migrate data from an Exchange server using either an Exchange administrator profile, or by entering the Exchange server’s credentials and creating a temporary Exchange administrator profile.
Read the Migrating to Zimbra Collaboration Server from Microsoft Exchange guide for complete migration details.
1. On the Source dialog box, enter the relevant Exchange server information:
•
Select Outlook Profile if you already have an Exchange administrator profile account, which allows you access to all other profiles you want to migrate.
•
Select Exchange Server if you do not have an Exchange administrator profile account and want to create a temporary one. Enter the Exchange server credentials.
•
Hostname/IP: Host name of the Exchange server name or IP address.
•
Admin ID. Name of the administrator account on the Exchange server.
•
Password. Password of the administrator account on the Exchange server.
An Exchange administrator profile is created for this migration. The profile is automatically deleted after the migration process.
2. Click Next.

Destination Server Configuration (Server)
This is the destination Zimbra server for data to be migrated. Enter your Zimbra server credentials.
1.
In the Destination dialog, enter the Zimbra server credentials.
•
Hostname. The name of the destination Zimbra server.
•
Admin Port. Port 7071 is the port the admin services run on.
•
Admin ID. The name of the administrator account on the destination Zimbra server.
•
Password. The password of the administrator account on the destination Zimbra server.
•
Use Secure Connection. Select the option for SSL secure communication with ZCS.

Selecting Users to Migrate
In the Users dialog box, select the users to migrate. The users are added to the Source Name and Destination Name fields.
You can upload user information using one of the population tools: Use Load CSV, Object Picker, or LDAP Browser, or add user information manually. After you have added users to migrate, click Next.
